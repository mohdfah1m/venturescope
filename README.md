# VentureScope

A VC sourcing and intelligence interface built with React + Vite, deployed on Vercel.

**Live:** `https://your-url.vercel.app` 

---

## Features

- **Companies** — search, filter by stage/sector/country, sortable table, bulk select
- **Company Profile** — overview, signals timeline, notes, save to list
- **Live Enrichment** — fetches summary, keywords, derived signals and sources per company
- **Lists** — create watchlists, add/remove companies, export as CSV
- **Saved Searches** — save and re-run named queries, persisted in localStorage

## Stack

- React + Vite
- No UI framework — pure inline styles
- Data seeded from `mock_companies.csv`
- Deployed on Vercel
